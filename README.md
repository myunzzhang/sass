# 🐸Partials(파샬)
--관련된것끼리 묶어서 분리 / 소스에 반복되는 부분들을 분리 분산시켜 모듈화 시키는 기능 

  * partials의 파일명은 _로 시작하며
  * 불러들일때는 @import '파일명',  이때 파일명에 _는 포함시키지 않고, 확장명도 포함시키지 않는다. 

💢 Scss는 _로 시작하는 파일은 컴파일하지 않는다.
